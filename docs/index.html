AlwaysUp 槓桿策略的市場背景

在回測期間（2024-2025 年），台灣股市適逢強勁的半導體與科技股單邊牛市。AlwaysUp 模型預測機率恆為 1.0，這會持續觸發 2x 槓桿持倉。在強烈單邊上升中，雙倍槓桿讓 AlwaysUp 取得了極高的累積報酬（平均超過 170%），但這也伴隨著雙倍的市場最大回撤與波動風險。如果股市反轉，2x 槓桿可能帶來永久性的資金重創。

交易確認與摩擦成本的影響

1-day 預測週期下，隨機策略因為高頻率進出（平均 9 次）而面臨劇烈的交易摩擦成本損耗（扣除 10 bps），最終產生 -51.6% 的悲慘平均報酬。然而，當使用 5-day 預測週期並搭配「連續兩日確認」機制時，隨機信號的過度換手被大大過濾，交易次數減半，使 Random 策略平均報酬提升至 +9.9%，充分證明了確認機制在控制交易磨損上的強大效用。

特徵工程與波動度特徵設計

本研究均使用 OHLCV（開盤價、最高價、最低價、收盤價、成交量）日資料，不直接使用絕對價格，而是轉換成技術指標與相對變化，以避免模型僅記住某一價格水位。

基礎技術特徵 (Base Features - 9 個)
return_1d 前一日到當日的日報酬率
ma_ratio 5日均線 / 20日均線，表示短期趨勢相對於中期趨勢的位置
bias_5 / bias_20 收盤價相對 5日與 20日均線的偏離程度
vol_chg 成交量日變化率，捕捉量能是否放大或縮小
rsi_14 14日相對強弱指標，衡量超買、超賣與動能狀態
macd / signal / hist 百分比形式計算的 MACD、訊號線與柱狀圖
波動與風險狀態特徵 (Volatility Features - 新增 4 個)
atr_ratio_14 14日 ATR / 收盤價，衡量近期每日價格震盪幅度相對股價的大小
volatility_20 20日日報酬率之標準差，描述近期收益率波動程度
volume_ratio_20 當日成交量 / 20日平均成交量，用以觀察量能是否異常放大
bb_width_20 20日布林通道寬度 / 中線，衡量價格震盪區間是否擴張
特徵設計的結論與啟示： 波動度特徵在 1-day 短線模型中效果有限，甚至會因為高頻雜訊讓平均報酬率略微下滑。然而，在 5-day 中期模型中，波動度特徵有助於模型識別「趨勢延續」與「高風險震盪區」的差異，使得 5d_vol 平均策略報酬大幅領先 5d_base（56.0% vs 43.1%），並成功打敗了多數個股的 Buy & Hold。

回測設定與指標說明

我們採用極為嚴苛且符合實戰情況的設定，以確保回測數據不含水分、具備高度實踐價值。

1. 時間序列切分

將 2018-2026 年資料清洗後，按順序切分為：
訓練集 (60%)：2018/01 - 2022/10，用於訓練模型。
驗證集 (20%)：2022/10 - 2024/05，用於調參、優化門檻。
測試集 (20%)：2024/05 - 2025/12，完全樣本外測試，得出最終報告指標。

2. 交易摩擦成本

本系統設定單次部位變化扣除 10 bps (0.10%) 的交易摩擦成本。每次調整持倉（如多頭轉平倉、或平倉轉 2x 槓桿）時，都會精確扣除成本。這也是 Random 短線策略會出現高額虧損、而 ML 模型能透過控制交易次數鎖定利潤的原因。

3. 最大回撤 (MDD) 與風險

最大回撤 (MDD) 代表帳面資產波峰至波谷的最大跌幅。AlwaysUp 策略雖然因大牛市槓桿取得極大報酬，但 MDD 往往超過 -50%（鴻海高達 -78.5%）。機器學習策略能將 MDD 大幅限縮，顯著優化了風險調整後報酬（Sharpe Ratio）。
